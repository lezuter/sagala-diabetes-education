**SAGALA** (Sadar Kandungan Gula) is an interactive educational web platform designed to raise awareness about **Diabetes Mellitus**. It provides comprehensive information regarding risks, prevention, and daily sugar consumption limits through an engaging, animated user interface.

## ğŸŒŸ Key Features

* **Interactive Learning Modules:**
    * **Diabetes Types:** Detailed explanation of Type 1 & Type 2 Diabetes.
    * **Risk Factors:** Infographics on modifiable and non-modifiable risk factors.
    * **Sugar Calculator:** Visual guide to sugar content in popular drinks and snacks (e.g., Boba, Instant Tea).
    * **Healthy Recipes:** curated recipes for healthy packed meals (e.g., Batagor Mini, Dorayaki Bayam).
* **Engaging UI/UX:**
    * Smooth **CSS Animations** (Fade-in, Slide-up, Bounce effects) for a dynamic reading experience.
    * Custom **Typing Effect** in the credits section.
    * Responsive layout optimized for various screen sizes, including specific media queries for mobile devices.
* **Visual Data Representation:** Uses intuitive cards and layouts to explain complex health statistics.
